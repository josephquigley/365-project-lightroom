# 365 Project — Lightroom Plugin

A Lightroom Classic plugin that shows a month-grid calendar view of a collection, with thumbnails and labels stating the day number on days with photos and empty cells on
days without. Helps photographers participating in a 365 Project visually identify missing days.

See [`CLAUDE.md`](./CLAUDE.md) for contributor setup.
See [`docs/superpowers/specs/`](./docs/superpowers/specs/) for design docs.
